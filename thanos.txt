===Thanos

Умеет дедуплицировать данные из нескольких промов
Алертинг компонент
Общий код с промом
20$ в месяц за террабайт в s3


Sidecar - зупускается рядом с prom и тащит данные в s3
Store gateway - ходит в s3 по запросам из query
Query - обрабатывает запросы от Api/grafana, есть webui, ходит в sidecar за 2часа
Сompact - сливает файлы в более крупные, sidecar сливает за 2 часа
Ruler - обрабатывает алерты

== Установка
Создать S3
data compaction - отключить в проме
Для каждого бакета настраивается compactor, gateway 

== Trouble 
Sidecar - прекратить запись в s3
compactor - из за разницы между compactor
