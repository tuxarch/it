=== Prometheus operator
- собирает метрики
- выполняет правила
- сохраняет данные в бд

Выполняются:
- съем метрик по таргетам
- обработка правил, по которым данные записываются в бд, либо алертится




Дашборд для графаны CoreDNS - 5926

Правила для алертменеджера пишутся в prometheusRules
Для сбора метрик в servicemonitor
