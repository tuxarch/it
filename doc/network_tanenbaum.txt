Компьютерные сети история:
1960 - Идея создание сети на основе коммутации пакетов, не реализованна AT&T
для армии США
1970 - ARPANET - бабушка сети, военная сеть охватившая университеты
1974 - TCP/IP - для соединения гетерогенных сетей. Потом TCP/IP был реализован
на сокете в университете Беркли
1980 - Появление DNS службы, отпочкование от военных

Телефон:
1982 - Аналоговый мобильный телефон
1991 - GSM 
2001 - 3G до 2 мбит/c c обменом данными или 380 кбит при движении

Беспроводные сети
OFDM - разделение спектра сигнала на фрагменты и паралельная передача по ним


Канальный уровень

Ethernet - 
Протокол ALoha: Пакетная передача в случайное время, доставка обнаруживается
путем обратной отправки посланного пакета от сервера, по своему серверному
каналу.
CSMA/CD - прослушивают линию когда нужно что-то отправить, и отправляют когда
линия свободна. При передаче слушают линию в поиске колизий, в случае таковый
подают сигнал об этом.


FastEthernet
GigabitEthernet
10G-ethernet


Wifi 802.11 
Клиент передает и получает данные через точку доступа
Несколько точек доступа могут быть соеденены (distribution system)

Произвольная сеть - аналог p2p для устройств (mesh)

Диапазон от 2.4 - 5 ГГЦ

OFDM - мультиплексирование. Данные передаются по 52 несущим из них 4 канала
синхронизации.
CSMA/CA - присутствует подтверждение о доставке кадра
NAV (вирутальный кадр) - устанавливает сколько линия будет занята

VLAN 
В кадре ethernet есть поле для VLAN идентификатора


Концентраторы - с одного порта передают на все
Мост - передает кадр в нужный пор, в соостветствии с таблицей
Широковещение мостов если не известно где адресат  и широковещание ip 
Мост в режиме прозрачности будет пересылать запросы на другие машины
Коммутатор - тот же мост, но нацелен на двухточечные линии

CDN - content delivery networks
Разделение контента по региональным серверам

Умный DNS - дает ответ для своего региона исходя из ip отправителя
