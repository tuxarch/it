Зона - это часть дерева доменных имен
Домен - имя ветви ответственной за узлы ns
TTL - время жизни записи
RTT - метрика для выбора днс-сервера
Блокировка - наложенные ограничения ресурсы в процессе работы
Транзакция - совокупность операций представляющие одно логическое целое.
Курсор SQL - значение указывающее на последний оператор
Тригер - хранимая пользовательская  процедура которая выполняется при работе с
данными.
Шардинг - разнесение данных на разные сервера
Блокчейн – это хронологическая база данных, т.е. такая база данных, в которой
время когда была сделана запись неразрывно связано с самими данными
Класс - абстрактный тип данных
wan - глобальная сеть
lan - локальная сеть
man - местная сеть
дуплекс - способ связи при котором данные могут передаваться в двух
направлениях
ОС - это программное обеспечение которое организует доступ к
ресурсам компьютера, а также предоставляет интерфейс для пользовательских
приложений
Компьютер - вычислительная машина
Алгоритм - последовательность действий
Мультиплексор - устройство имеющее несколько входов и один выход. Уплотнение
канала связи.
Программа - реализованный и запущенный на вычеслительной машине алгоритм
itil - менеджмент в области информационных технологий
интерфейс или протокол - правила взаимодействия между объектами
стек протоколов - полный набор правил описывающий взаимодействие компонентов
необходимый для работы сети
концентратор (хаб, повторитель) - дублирует электрический сигнал
коммутатор(свич,мост) - на уровне канальном и транспортном имеет таблицу arp
маршрутизатор - на сетевом еще
pass - платформа как сервис - совместный стек приложений и ос
saas - приложение как сервис - провайдерское по - пользовательские только
данные
iaas - инфраструктура как сервис - пользовательское приложение
rest - архитектурный стиль взаимодействия в распределенных вычислениях. Есть
запрос GET к клиенту и есть параметры в этом запросе
