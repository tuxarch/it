float: left; - элемент переносим влево (режим обтекания)
width: 65px; - ширину ставим
min-height: 50px; - минимальная высота
margin-right: 9px; - для позиционирования элемента

clear запрещает обтекание элемента другими элементами. Вот его значения:
    left — запрещено обтекание слева;
    right — запрещено обтекание справа;
    both — запрещено обтекание с обеих сторон;
    none — обтекание разрешено.

 .clearfix::after { - используем специальный класс для создания колонок
     display: table;
     content: "";
     clear: both;
     }
