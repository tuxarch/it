Код, написанный на языке препроцессора, не используется прямо в браузере, а
преобразуется в чистый CSS-код с помощью специальных библиотек. 
Три самых известных препроцессора — это LESS, SASS и Stylus


@name value - задаем переменную
color: spin(red, 90) - позволяет проворачивать цвета, создавая мягкий переход
цвета

Позволяет применять вложенность

___
super-button {
   &-red { color: red;  }
   &-blue { color: blue;  }}
 
Можно быстро управлять похожими  классами super-button-red
					  super-button-blue

_________
Позволяет создавать аналог hover при наведении

color: lighten(red, 50%); // светлее red на 50%
color: darken(blue, 25%); // темнее blue на 25%
color: saturate(green, 20%); // green насыщеннее на 20%
color: desaturate(blue, 50%); // blue бледнее на 50%
