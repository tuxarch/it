Нас интересует колонка raw_value

Raw Read Error Rate - частота ошибок при чтении данных с диска, происхождение которых обусловлено аппаратной частью диска.
Spin Up Time - время раскрутки пакета дисков из состояния покоя до рабочей скорости. 
Spin Up Retry Count - число повторных попыток раскрутки дисков до рабочей скорости, 
Seek Error Rate - частота ошибок при позиционировании блока головок. 
Reallocated Sector Count - число операций переназначения секторов.
