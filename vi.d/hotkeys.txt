=== hotkeys
db - удаляет слово слева
I/A = ввод с сначала/конца строки
dG/dgg - удалить от курсора и до конца/начала файла
ce - удалить от указателя до конца слова
yw - копировать одно слово
O - создать строку выше предыдущей
cc - удалить строку и начать редактирование
CTR+y, CTR+d Перемотка вверх/вниз без движения курсора
H/M/L - перемещаемся по странице 

=== commands
%s/old/new/g - замена во всем файле
:! - выполнить в shell
:r !ls - выполнить а результат отправить в файл
:e file - создать новый файл/ редактировать
:w file - сохранить в файл
редактируем через сеть:
:e &lt;scp|ftp|ftps&gt;://user@host/path/to/the/file.txt


=== custom keys
F1 - easymotion
F3 - отключить autopair
F4 - проверка орфографии
F6 - просмотр markdown
F8 - нумерация строк
F7 - линтинг
F9 - запуск python скрипта
Ctrl+n - дерево
Ctrl+m - меню с плагинами
,+<char><char> - запуск esymotion


=== nnn
Ctrl+n - показать дерево
ctrl+g - закрыть дерево
Файл открывается по Enter при выходе все закрывается 
ctrl+t - открыть файл в новом табе
ctrl+x - открыть в нижнем сплите
gt - переключение между табами
ctrl+w ctrl+w - переключение между вкладками
