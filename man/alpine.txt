setup-alpine
юзаем ключ -с -f для создания конфига установки

Используем переменную 
SWAP_SIZE=0 

Поставляется с syslinux
Напрямую правим /boot/extlinux.conf без перезборки

разрешение консоли задаем через vga=792 для ядра

Для выключения используем poweroff

Обновление:
apk update && apk upgrade
