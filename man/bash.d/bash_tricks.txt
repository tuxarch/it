хитрости:
ставишь проблем перед командой и она не сохраняется в истории
!! - выполнить предыдущую команду
sudo !! - запустить команду последнюю
cp ExampleFile old-!# - скопировать бекaп
view - аналог vi, но только для чтения
lftp, ngrep, iftop
CTRL+x ctrl+e  - редактировать команду в редакторе
alt+\ - удаляет пробелы
cx+cv - версия баша
/etc/inputrc - конфиг реадлайна

bind -p (-l) - посмотреть доступные бинды

set -o vi - переключить bash в режим vi {ECS, I}
set -o emacs - вернуться в дефолт режим
