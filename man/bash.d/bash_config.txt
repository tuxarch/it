/etc/shells - показывает какие командные оболочки установленны в системе.
chsh -s /bin/sh - переключениe командной оболочки

каталог /etc/bash - хранит глобальные насройки для шелл.
	bashrc - хранит глобальные настройки
	bash_history - показывает историю введенных команд
	bash_logout - используется при закрытии оболочки

Программа login [/bin/login] обращается к файлу /etc/passwd и затем обращается к файлу настроек /etc/profile

В домашней папке также есть эти файлы которые позволяют переопределить глобальные настройки. bashrc - хранит настройка не для интерактивной оболочки.

Сначала отрабатывает файл /etc/profile затем он грузит файл /etc/bashrc, а тот подхватывает еще кучу файлов отвечающих за отображение файлов, папок и тд.
