Можно также воспользоваться ltrace, gdb они тяжелее, требуют компляцию
приложений с отладкой.

Strace - заставляет программу выполнять действия дважды, один раз для
отладчика.

strace -o file.log -s 512 htop - отправить в файл со строкой 512 (32 по умолч)

strace -c command - используем таблицу обращения к системным вызовам

strace -p PID - отладка запущенного процесса

echo $? - завершилась ли программа с ошибкой
