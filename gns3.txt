Cостоит из двух пакетов, клиента и сервера.
Сервер подцепляется по сети, по порту 3080.

Для работы с образами сisco используем dynamips позволяющую моделиривать
оборудование без лишних нагрузок.
Для этого смотрим требования к RAM и Idle-PC - смещение для образа которое
отрубает оверхед. Оно вычисляется автоматически.

Рекомендуемые образы:
c3640, c3660, c3725, c3745 and c7200 IOS

Для свичей нет полноценного моделирования в dynamips на канальном уровне.
Используем IOU L2 image or vIOS L2 images with Qemu.
